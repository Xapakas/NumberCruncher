Using the Reduce Method

While these were calculated by hand, for tougher problems we can use the Reduce Method to make things easier. The reduce() operation takes an array and passes each element of it through whatever function is inside the parentheses. For example, our array:

2,2,4,6,6,7,8,11,12,18,19,20,23,24,29,31,41,43,43,50

can be saved as "var array" on JavaScript. We can then take that array and use the reduce() operation to find its sum. The button below me will run the line of code

document.getElementById("here").innerHTML = array.reduce(getSum)

in which "here" is the ID of the line that comes after it. In this, getSum is our function that takes each number of the array and adds it until we get our final total.

Push the button!

Great! You've just found the sum of all the numbers in our array. Because there are 20 elements in our array, the final calculation is fairly simple.

399 / 20 = 19.95
